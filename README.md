# YPDouYinLikeAnimation（QQ打趣群：68153279）
## ☆☆☆ “iOS仿抖音点赞动画” ☆☆☆

### 应用场景
```
近期项目中有需要类似抖音小视频的双击点赞的动画，简单仿照了一下，有需要的同学可以直接导入工程中使用
```

### 具体用法
```
获取单例对象：YPDouYinLikeAnimation *ani = [YPDouYinLikeAnimation shareInstance];
实现动画：[ani createAnimationWithTap:tap];
tap：触发动画对应的手势
```
